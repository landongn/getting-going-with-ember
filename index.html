The Basic Pitch

Flexible, Declarative, Composable API.

Mixins, forward-compatable web components.

(if you use ember-cli) write ES6 JS today.

Dedication to sane depreciations, RFC process.

Constantly Improving.

Behavioral actions API.

Writing ember fast makes you feel like a superhero.


- No focus on build tools, no focus on compilers, no focus on testing integration. It's the most batteries included framework with ember-cli
- jQuery, Handlebars (Htmlbars), RSVP, Babel, node, etc
- ES6 opens up doors for proper modularization, getting rid of quirks and tribulations of things like r.js, browserify,

The Router

The most important element of ember.

Defines states the application can react to.

Example Routing Definition


// app/router.js
Router.map(function() {

  // defines a standalone route with no substates or arguments
  // routable as 'stateA'
  this.route('stateA');
  
  // defines a standalone route with no substates and an argument
  // routable as 'stateB'
  this.route('stateB', {path: '/stateB/:someArgument'});
  
  // defines a route with a child state
  // routable as superState.loading, superState.error,
  // superState.index, superState.childState
  this.route('superState', function () {
  	this.route('childState');
  });

});
